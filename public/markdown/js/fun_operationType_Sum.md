# js内置方法的两种不同操作方式

## 1.直接操作调用对象 ##


 - string.split()
 - array.pop() array.push()
 - array.shift() array.unshift()
 - array.splice()  array.join()
 - array.sort() array.reverse()
 - ...

## 2.操作调用对象的副本 ##

 - string.replace()
 - string.substr()
 - string.substring()
 - string.toUpperCase()
 - string.toLowerCase()
 - string.slice()   array.slice()
 - string.concat() array.concat()
